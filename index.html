Welcome to AMReX’s documentation¶

AMReX is a software framework library containing all the functionality to write massively parallel, block-structured adaptive mesh refinement (AMR) applications. AMReX is freely available at https://github.com/AMReX-Codes/amrex. The most current version of this User’s Guide can be found in the  git repository at AMReX/Docs/AMReXUsersGuide.

The copyright notice of AMReX is included in the amrex directory as README.txt.
