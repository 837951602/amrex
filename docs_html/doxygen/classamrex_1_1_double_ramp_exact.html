Detailed Description

Double sphere test case Test case where f(x,y) = 2*(x-0) + (y - 1) g(x,y) =-2*(x-1) + (y - 1) in the input file: ramp_normal_0 = 2. 1. 0. ramp_normal_1 = -2. 1. 0. ramp_point_0 = 0. 1. 0. ramp_point_1 = 1. 1. 0.
